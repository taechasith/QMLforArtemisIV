# Release Notes v0.3.0

Status: Accepted release package
Decision: D033-C
Date: 2026-07-14

## Allowed Claim

This release supports one public claim: under the P001 frozen public-data,
development-only benchmark, the tested QML candidates did not outperform strong
classical controls and no QML candidate qualified for a Gate 6 mission
experiment.

## Boundary

The release is negative benchmark evidence only. It does not claim quantum
advantage, NASA approval, flight readiness, mission-loop validation, locked-data
validation, or a proven new QML invention. Calibration and final-test data
remain locked. Gate 6 remains unauthorized.

## Included Evidence

- Gate 5 official technical `FAIL`.
- Gate 5X exploratory Q01b and FQK negatives.
- Classical-first and recall-first follow-up evidence labeled for future
  research only.
- Manuscript Results/Discussion draft and claim-boundary table.
- Release-support model, simulator, data, and limitation cards.
- Clean-clone reproducibility correction and release-candidate manifest.

## Not Included

- DOI minting.
- Released trained model weights.
- Mission-loop experiment.
- Calibration or final-test evaluation.
- Hardware/GPU quantum run.
- Gate 6 protocol execution.
