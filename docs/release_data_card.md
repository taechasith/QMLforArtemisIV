# Release Data Card

Version: 0.1.0
Decision: D028-C
Prepared: 2026-07-14
Status: Release-support card; public-data benchmark record

## Data Scope

The release-support record covers public-source provenance, frozen splits,
development-only model evidence, exploratory post-Gate-5 evidence, and
paper-ready reporting artifacts.

## Locked Data Boundary

Calibration and final-test access remain locked. Gate 6 mission-loop data and
mission-owned evidence are not part of the released evidence package.

## Intended Use

Use the data record to reproduce the public-data development benchmark and to
audit the negative QML result and post-Gate-5 exploratory negatives.

## Prohibited Use

Do not use the public-data development benchmark as final mission validation,
flight evidence, or proof of fuel savings.

## Release Boundary

D028-C authorizes no new data access, final-test read, calibration read,
mission-loop read, or Gate 6 run.
