# Release Model Card

Version: 0.1.0
Decision: D028-C
Prepared: 2026-07-14
Status: Release-support card; no trained model released

## Model Release Status

No trained model is released by the current package. The project reports model
comparisons and negative evidence only.

## Evaluated Model Families

The preregistered Gate 5 Q01 quantum-kernel candidate failed against the C06
physics-residual control. Post-Gate-5 Q01b and FQK were valid exploratory
negatives. CSAFE-RF recall-first evidence selected a classical logistic
candidate for future safety-objective design discussion only.

## Intended Use

The model evidence may be used to understand which tested QML approaches did
not outperform strong classical controls under the frozen public-data
development benchmark.

## Prohibited Use

Do not use any reported model as a mission-control component, safety filter,
flight model, operational planner, or QML advantage demonstration.

## Release Boundary

D028-C authorizes no model release, Gate 6 run, calibration/final-test access,
mission-loop execution, QML invention claim, quantum-advantage claim, or Gate 5
reinterpretation.
